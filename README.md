# QA Automation - ATMosphere

## Descripción
Proyecto de automatización de pruebas para ATMosphere 2.0.

## Objetivo
Validar funcionalidades del sistema mediante pruebas automatizadas.

## Tecnologías
- Java
- Maven
- JUnit

## Ejecución local
Para ejecutar las pruebas:

mvn clean test

## Estructura del proyecto
- src/test/java → pruebas
- src/main/java → lógica base

## CI/CD
Pipeline preparado para ejecutarse en GitLab CI

## Estado
Proyecto en fase inicial de configuración
