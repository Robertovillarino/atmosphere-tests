# Test Plan - ATMosphere 2.0

## 1. Objetivo
Validar las funcionalidades principales del sistema ATMosphere 2.0 asegurando su correcto funcionamiento antes de liberación.

## 2. Alcance
Este plan cubre pruebas funcionales iniciales basadas en la referencia de ATMosphere 1.0.

## 3. Módulos a probar
- Login
- Navegación
- Operaciones principales (pendiente por definir)

## 4. Tipos de prueba
- Smoke (validación rápida)
- Funcionales
- Regresión (a futuro)

## 5. Estrategia
Se diseñarán casos de prueba basados en flujos críticos identificados en la versión 1.0.

## 6. Herramientas
- Java
- Maven
- JUnit
- GitLab CI/CD (en proceso)

## 7. Estado actual
- En fase inicial
- Sin ambiente de QA disponible
